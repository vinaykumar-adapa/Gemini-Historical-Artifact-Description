## üöÄ Project Overview

The **Gemini Historical Artifact Description** app is a multimodal Generative AI application. It allows users to explore history by either entering the name of an artifact or uploading an image. Using **Gemini 1.5 Flash**, the app identifies the object and generates a high-quality, scholarly blog post or description based on the user's preferred word count and historical era.

---

## ‚ú® Features

* **Multimodal Search:** Use text or images to identify artifacts.
* **Smart Word Count:** Generate content from 100 to 1500+ words.
* **Educational Engagement:** Displays random historical facts while processing.
* **One-Click Export:** Copy to clipboard or download as a `.txt` file.
